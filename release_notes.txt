Cập nhật v1.0.5:
- Nâng cấp phiên bản hệ thống lên v1.0.5 để tạo bản phát hành mới.
- Chuẩn hóa các bộ lọc và nhãn trạng thái theo đúng API của OTruyen (Đang phát hành, Hoàn thành, Sắp ra mắt, Truyện mới).
- Hỗ trợ phân trang cuộn mượt mà theo đúng số lượng dữ liệu trả về tự nhiên từ API của OTruyen.
- Tích hợp bộ lọc Trạng thái và Năm phát hành vào trang chi tiết Thể loại tiện lợi.
- Thiết kế lại trang chủ với thanh TabBar 3 mục (Mới cập nhật, Truyện mới, Theo dõi) và giao diện gọn gàng.
- Sửa lỗi quyền cài đặt ứng dụng cập nhật (REQUEST_INSTALL_PACKAGES) mượt mà trên Android.
